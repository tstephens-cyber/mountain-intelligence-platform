# Mountain Intelligence

## Mission
Build an AI-powered executive operating system that helps business leaders make faster, smarter decisions.

## Vision
Mountain Intelligence combines AI coaching, financial intelligence, document analysis, forecasting, KPI monitoring, and operational management into one platform.

## Core Modules
- Executive Dashboard
- AI Coach
- Document Intelligence
- Financial Intelligence
- Analytics
- Workspace
- Action Center
- Knowledge Base
- Settings

## Technology Stack
Frontend:
- React
- Vite
- TypeScript
- Tailwind CSS

Backend:
- Node.js
- Express

AI:
- OpenAI Responses API

Database:
- PostgreSQL

Hosting:
- Render

Version:
5.0.0 Foundation

## Development Principles
- Mobile first
- Production quality
- Reusable components
- Strong TypeScript typing
- Clean architecture
